How to Use

Get started in just a few simple steps

1

Download DNS JoinV

Download the DNS-JoinV.cmd file from GitHub releases. No installation required!

2

Run as Administrator

Double-click the file and click "Yes" when prompted for administrator privileges.

3

Select Your Network Adapter

Choose your active network adapter (Ethernet or Wi-Fi) from the dropdown.

4

Choose DNS & Apply

Select your preferred DNS provider and click "Apply DNS". That's it!
